#### What is this site?
This site serves as my personal portfolio, where I can showcase my projects, certifications, and potentially any scientific papers I publish in the future. It also acts as an easy place to keep my resume accessible to recruiters.

#### More info about this site.
This is the third iteration of the site, and by far the best-looking one yet! The site is built with Eleventy (11ty), Webpack.js, Font Awesome, and Bootstrap CSS. Eleventy powers the site’s structure, while Webpack bundles all the JavaScript modules. I use Font Awesome for the social media icons in the footer, and Bootstrap provides the overall styling of the site.
