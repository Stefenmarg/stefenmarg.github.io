My first project of 2024 is this site which i host on Github with Github pages and uses Jekyll which is a static site
generator that allows you to create simple, static websites without the need for a database or server-side processing 
and Liquid which is a templating language that is used by Jekyll to generate the static website.

It is build on the default Jekyll template and uses utilities like font-awesome for rendering the social media icons
in the footer of the webpage.
