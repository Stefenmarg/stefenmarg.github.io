What is the purpose of this site?

This site will primarily function as a personal portfolio to highlight my projects and any scientific papers I may publish. Additionally, it will serve as a repository for my resume and other certifications, making them easily accessible to recruiters.

All the projects featured here are created during my free time. While tips are not required, they would be greatly appreciated as a gesture of support!

Information about this site.

In the backend before the site is built into a static site we are using Eleventy (11ty) that has recently joined Font Awesome the service that I use for the social media icons on the footer of the site. For bundling JavaScript code we are using Webpack that bundles all the code in one neat and compact file. Additionally, this site has a repository that can be found here and it has everything that you would need to make a site for yourself.

This site is using cookies for the site theme, the snowflakes during the holidays and analytics collection for the site. You can see the Cookie Policy here!
