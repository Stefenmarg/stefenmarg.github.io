# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] -- 2024-03-07

### Added
 - Root directory and files generated by `jekyll build ./`
 - Font awesome resources.
 - Social media & menu settings
 - Social media icon rendering for each account
 - Text for the purpose of the site.
 - A .gitignore file to leave out of the repo the cache files for jekyll building.
 - A limit of the 5 most recent posts to the home page.

### Changed
 - The way that social media accounts would get added to list
 - The way that menu buttons get added to list.
 - The use of decription for metadata responce.
 - Where the RSS feed button is placed from body to navigation menu.
 - Split the main part of the website to 2 columns to have in one the content from index.md and in the second one to have the profile picture of the site's owner.

### Removed
 - The 'About me' page due to not having a use for now.

### Fixed
 - CSS Issues where some objects would not get centered correctly and would drift to the left.
